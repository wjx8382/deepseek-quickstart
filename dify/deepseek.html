请帮我写一个Python函数，计算斐波那契数列的第n项
当然，这是一个计算斐波那契数列第n项的Python函数：

def fibonacci(n):
    if n <= 0:
        return 0
    elif n == 1:
        return 1
    else:
        a, b = 0, 1
        for _ in range(2, n+1):
            a, b = b, a + b
        return b
这个函数的时间复杂度是多少？
这个函数使用迭代方法，时间复杂度为O(n)，空间复杂度为O(1)。如果您需要更高效的方法，我可以为您提供使用矩阵幂运算的O(log n)解法。
